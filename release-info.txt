SuperGenPass Chrome Session 

----------

Stores multiple SuperGenPass style master passwords for a chrome session.

----------

This extension stores multiple SuperGenPass style master passwords for a chrome session. It performs almost all of the original SGP bookmarklet functionalities, in a secure way.

2.0 - 20100316

- The domain part used in each tab is shown, and can be edited for the session
- Minimal options page: Allows blocking or unblocking a domain, or all, from auto-filling the password on page load
- Shows the last used password for each tab

1.2 - 20091228

- Stores multiple passwords for a domain (0 is default, but you can apply any latter) -
